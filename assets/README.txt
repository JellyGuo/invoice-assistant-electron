📦 使用说明：invoice-assistant-electron

本目录包含以下内容：

1.  双击 invoice_assistant.dmg 安装 application

2. 🔧 start.sh  - 避免苹果安全隔离
   - 第一次使用时运行本脚本  
   - 自动解除 macOS 的安全隔离，并启动 App  
   - 使用方法：
     打开终端，cd 到 App 所在目录，运行：
     
         ./start.sh

3. 🛑 stop.sh   - 避免应用非正常关闭存在孤儿进程
   - 正常情况下，关闭窗口并未退出应用，右击 Quit 会完全退出
   - 如果想避免非正常关闭下存在孤儿进程，可以运行此脚本
   - 关闭后台监听端口（8123）以防止残留服务  
   - 在你关闭应用后，若发现端口仍被占用，可以运行：
     
         ./stop.sh

4. 🚀 invoice-assistant-electron.app  
   - 主应用目录，双击运行即可（推荐使用 start.sh 第一次运行）

5. 所有上传文件、中间状态的临时数据都保存在应用内部，删除应用后全部删除

🔒 macOS 安全提示：
如果系统阻止启动，请“右键 → 打开”以绕过。
