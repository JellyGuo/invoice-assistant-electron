📦 使用说明：invoice-assistant-electron

本目录包含以下内容：

1. 🔧 StartApp
   - 第一次使用时运行StartApp启动应用 

2. 🚀 invoice-assistant-electron.app  
   - 主应用目录，双击运行即可（推荐使用 StartApp 第一次运行）

3. 所有上传文件、中间状态的临时数据都保存在应用内部，删除应用后全部删除

4. 文件命名规范
   - 火车票: "train-上海-北京.pdf"
   - 打车发票: "上海打车-日期-xxx-金额-xxx.发票.pdf"
   - 打车行程单: "上海打车-日期-xxx-金额-xxx.行程单.pdf"
   - 餐费发票: "餐费-日期-xxx-金额-xxx-晚餐.pdf"
   - 住宿费发票: "北京-住宿费-入住天数-2天-发票.pdf"
   - 住宿费水单: "北京-住宿费-入住天数-2天-水单.jpg"
